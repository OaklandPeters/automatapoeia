Shell Commands
----------------
To run full build process, which includes running the TypeScript compiler:
```shell
gulp
```

Run Typescript compiler:
```shell
cd automatapoeia
tsc
```

Run unit-tests in `test/`:
```shell
npm test
npm test src/cats/test/
```
